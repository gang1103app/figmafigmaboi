## Key Features Highlighted

### Visual Indicators
- **Plant Health Bar**: Color-coded (green=healthy, yellow=warning, orange=critical, red=danger)
- **Golden Plants**: Special golden border and âœ¨ icon
- **Free Items**: ğŸ icon next to free items
- **Active Background**: Green checkmark and border
- **Placed Plants**: Semi-transparent with "âœ“ Placed" status

### Interactive Elements
- **Drag and Drop**: Plants can be dragged from inventory to garden
- **Remove Button**: Appears on hover over placed plants
- **Purchase Limits**: Shows "Max owned" when limit reached
- **Seed Affordability**: Buy button disabled when insufficient seeds

### Color Scheme
- Background: Dark gradient (from #071021 to #0e1723)
- Primary accent: Brand primary color
- Health bar colors: Green/Yellow/Orange/Red based on health level
- Golden plants: Yellow/amber tones with gradient background
- Cards: Semi-transparent slate with backdrop blur

## Responsive Design
- Grid layout adjusts for mobile (2 cols) and desktop (4-5 cols)
- Touch-friendly drag and drop
- Readable text sizes across devices
